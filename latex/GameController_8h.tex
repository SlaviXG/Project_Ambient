\doxysubsection{Detailed Description}
This header file contains Game\+Controller, Game\+Interactor and Cell\+Interactor classes. 

Game\+Controller class manages the game logic and updates the graphical representation of the game. Game\+Interactor and Cell\+Interactor are interfaces for interaction between the controller and the view and logic layers, respectively.
