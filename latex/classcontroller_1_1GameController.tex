\doxysubsection{Detailed Description}
\mbox{\hyperlink{classcontroller_1_1GameController}{Game\+Controller}} class is responsible for managing the game logic and updating the graphical representation of the game. 

\mbox{\hyperlink{classcontroller_1_1GameController}{Game\+Controller}} inherits from QObject, \mbox{\hyperlink{classcontroller_1_1CellInteractor}{Cell\+Interactor}} and \mbox{\hyperlink{classcontroller_1_1GameInteractor}{Game\+Interactor}}, and encapsulates game logic, interactions with different parts of the application like views and environment, and provides an interface for generating random cells and controlling the game.
