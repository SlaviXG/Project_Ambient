Detailed Description

GameController class is responsible for managing the game logic and updating the graphical representation of the game.

GameController inherits from QObject, CellInteractor and GameInteractor, and encapsulates game logic, interactions with different parts of the application like views and environment, and provides an interface for generating random cells and controlling the game.
