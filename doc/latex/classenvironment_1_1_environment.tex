\doxysubsection{Detailed Description}
This class represents the environment in which cells live. 

The \doxylink{classenvironment_1_1_environment}{Environment} class is responsible for managing the cells and the frames in which they live. It provides methods for adding and removing cells, updating their positions, and performing various other operations related to the environment.
