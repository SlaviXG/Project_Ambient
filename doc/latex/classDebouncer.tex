\doxysubsection{Detailed Description}
A class that provides a debounce functionality. 

This class provides a way to ensure that a function or method is not called more frequently than a specified timeout. This can be useful in scenarios where a function is triggered by an event that can occur rapidly (such as a button press), but the function itself should not be executed too frequently.
