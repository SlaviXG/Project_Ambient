\doxysubsection{Detailed Description}
\doxylink{classcontroller_1_1_game_controller}{Game\+Controller} class is responsible for managing the game logic and updating the graphical representation of the game. 

\doxylink{classcontroller_1_1_game_controller}{Game\+Controller} inherits from QObject, \doxylink{classcontroller_1_1_cell_interactor}{Cell\+Interactor} and \doxylink{classcontroller_1_1_game_interactor}{Game\+Interactor}, and encapsulates game logic, interactions with different parts of the application like views and environment, and provides an interface for generating random cells and controlling the game.
